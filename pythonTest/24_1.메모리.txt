메모리 

data segment -----

global(static) 영역 - 전역
-> 프로그램 시작 후 종료 전까지 변수 등을 저장
-> 이름만 저장할 뿐 값은 heap 영역에 저장됨

heap 영역 - 데이터의 객체(숫자, 문자, 함수 등)가 저장됨
-> reference count로 관리가 되며 0이 되면 삭제 됨
-> 모든 값은 heap 영역에 저장 됨
-> 함수 또한 객체로서 저장됨

stack 영역 - 지역
-> 함수 등 실행 시 함수 내의 내용이 저장되는 영역
-> 값은 heap 영역에 저장 되며, 함수가 종료 되어 stack 내에 변수가 삭제 될 경우
연결된 heap의 값의 reference count가 0이면 해당 객체도 삭제

code segment -----

기계어 저장 (함수 정의, 루프, if문 등)
text로 언어적으로 저장
